El OrinocoDEV, tiene como meta, generar una base de datos de los proyectos en las diferentes fases que se encuentren para apoyarlos a través de rondas de financiamiento, organizadas a futuro por las diferentes organizaciones miembros de la comunidad, así mismo, tener a mano a los diferentes desarrolladores, diseñadores gráficos, emprendedores, inversionistas y publico interesado en tecnología, para crear una base de datos abierta acerca de quienes son, en que trabajan, que han logrado y como contactar con ellos para trabajos como “freelance”.

Está dirigido a toda persona que sienta curiosidad por la tecnología, estudiantes, empresarios, emprendedores, diseñadores gráficos y cuerpo docente de las instituciones educativas del territorio nacional.

Es un espacio para que los emprendedores, diseñadores gráficos y desarrolladores de software puedan interactuar entre ellos, construir proyectos productivos y/o compartir conocimientos con el publico en general asistente a los eventos, así mismo las pequeñas y medianas empresas (Startup) que tengan proyectos podrán:

✗ Mostrar los productos y/o servicios a través de la plataforma del evento.

✗ Realizar una base de datos de sus potenciales clientes.

✗ Mostrar tecnologías actuales para el desarrollo de software y diseño gráfico.

✗ Generar crowfunding para que los proyectos sean apoyados por cualquier integrante de la sociedad.

✗ Creación de perfiles para desarrolladores y diseñadores gráficos que le permitan a estos mostrar sus trabajos y el dominio de herramientas.
